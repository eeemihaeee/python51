ГЕНЕРАЛЬНАЯ И ВЫБОРОЧНАЯ СОВОПНУСТИ. ВИДЫ ВЫБОРКИ. СПОСОБЫ ОТБОРА.
Термины:
Генеральной совокупностью называют совокупность однородных объектов,
относительно некоторого качества или количественного признарка.

Выборочной сокупностью называется совокупность объектов, 
отобранных из генеральной совокупности(выборка).

Объем совокупности - число объектов, выбранных для исследования.
Например из 1000 деталей отобрано 100,
то объем генеральной совокупности N = 1000, а объем выборки n=100.

Повторной называют выборку, при которой отобранный объект в
генеральную совокупность возвращается.

Бесповторной - генеральную совокупность не возвращается.

На практике применяют различные виды отбора (2х типов):
1) Отбор, не требующий расчления генеральной совокупности на части.
(простой случайный бесповторный отбор, простой случайный повторный)
2) Отбор, где генеральная совокупность разбивается на части.
(типический отбор, механический отбор, серийный отбор)
ТИПЫ ОТБОРОВ:
1. Простой случайный - объекты извлекают по одному с 
	помощью повной и бесповторной выборки. 
2. Типический - объекты выбираются не из всей генеральной совокупности,
	а из ее "частей" типичной выборки.
3. Мехнический - ген. совокупность делят на столько групп,
	сколько объектов должно войти в выборку, а из каждой группы 
	выбирают лишь 1.
4. Серийный - выбираются объекты не по одному, а "сериями",
	кот. подвергаются сплошному обследованию.

ЭЛЕМЕНТЫ ТЕОРИИ КОРРЕЛЯЦИИ.
 Функциональная, статистическая, корреляционная зависимости.
 
Зависимость между переменными X и Y называется функциональной,
если существует функция y = f(x), по кот. каждому значению
х из множества X ставится в соответствии единственное значение
y из множества Y.
Однако, не всякую зависимость можно представить в виде функции.
Иногда одному фиксированному значению Х соответствует множество
Y.
--> Статисческой называют зависимость, при кот. изменение одной 
из величин влечет измение распределения другой. В частности,
статистическая зависимость проявляется в том, что при 
измении одной из величин меняется среднее значение другой.
В этом случае статистическую зависимость называют корреляционной.
Например, если при x1 = 2 величина Y приняла значение y1= 5,
y2 = 6, y3 = 10, то условное среднее y(sr x1) = (5+6+10)/3 = 4.
Аналогично определяется и условное среднее X (x(sr y)).
-->Условно средним x(sr y) называется среднее значение
арифимитеское наблюдение значений Х, соответветствующих Y = y.

Уравнение y(sr x) = f(x) называется выборочным уравнение регрессии.
f(x) - выборочной регрессией, а ее график - выборочной линией регрессии.
fu(x) - выборочной регрессиией, если х(sr y) = fu(y).

ОСНОВНЫЕ ЗАДАЧИ КОРРЕЛЯЦИИ.
1. Установить форму корреляционной связи, т.е. вид функции регрессии
(линейная, квадратическая, показательная и пр.)
В случае, если обе функции f(x) fu(x) линейны, то корреляцию
называют линейной, в противном случае - нелинейной.
2.Оценить тесноту(силу) корреляции.
Теснота корреляционной связи оцениваетсяпо величине рассеяния
значений Y вокруг условного среднего y(sr x). Большое рассеяние
показывет слабое значение зависимости либо об ее отсутствии.

Уравнение прямой линии регрессии.

Пусть требуется найти теорическое уравнение Y(sr x) = f(x,a1..an)
регрессии Y на Х. Параметры a1...an находятся 
методом "наименьших квадратов".
f(X, a1...an) = a1*x + a2
Cоставим функцию, зависящую от параметров a1, a2:
F(a1;a2) = SUM(Yi - yi)^2 = SUM(a1*xi + a2 + yi)^2
Для отыскания минимума данной функции,
приравняем к нулю частные производные:
F'a1 = 2*SUM(a1*xi + a2 - yi)*xi = 0
F'a2 = 2*SUM(a1*xi + a2 - yi) = 0
Получаем систему уравнений:
|--> a1 * SUM(xi ^2) + a2*SUM(xi) = SUM(xi * yi)
|
|--> a1 * SUM(xi) + a2*n = SUM(yi)
Воспользуемся след. тождествами:
x(sr) = SUM(xi)/ni
y(sr) = SUM(yi)/ni
x^2(sr) = SUM(xi^2)/ni
y^2(sr) = SUM(yi^2)/ni
x*y(sr) = SUM(x*y(sr)/ni
Подставим полученные выражения 
в систему и разделим каждое уравнение на n
|--> a1 * x(sr) + a2*x(sr) = x*y(sr)
|
|--> a1 * x(sr) + a2 = y(sr)
Решением системы являются коэффициенты:
a1 = (xy(sr)-x(sr)*y(sr))/Q^2
a2 = y(sr) - a1*x(sr)

ВЫБОРОЧНЫЙ КОЭФФИЦИЕНТ КОРРЕЛЯЦИИ.
--> rb признаков Х и Y называется число ,равное среднему
геометрическому коэффициенту регрессии и имеющее их значение:
rb =( xy(Sr) - x(sr)*y(sr) )/ Qx*Qy
Коэффициент кореляциии нужен для 
оценивания тесноты корреляционной связи. Имеет след. свойства:
1) |rb| <= 1 или -1<=rb<=1
2) Условие |rb| = 1 является необъодимым и достаточным условием
существования лтнейной функциональной зависимости.
3. При rb = 0 линейной корреляционной связи между признаками
не существует.
О тесноте свяхи можно судить по значению коэффициента корреляции
используя шкалу Чеддока:
